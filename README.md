# Cameltest

## Description.

Test of Camel 
* Textfile as input
* Logging headers (key, value) to logfile
* Textfile as output.

## How to build locally
```
cd to directory where pom.xml (root pom) is locaded
mvn clean package 
```

## How to run locally
```
cd to taget directory
java -jar cameltest-1.0.0-SNAPSHOT.jar
```
